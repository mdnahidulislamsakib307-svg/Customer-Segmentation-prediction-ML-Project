# 🛍️ Customer Segmentation Prediction (Machine Learning Project)

---

## 🔍 Problem Statement

Businesses often struggle to understand different types of customers and their spending behavior. Without proper segmentation, it becomes difficult to target the right audience, personalize offers, and maximize profit.

This project aims to segment customers into different groups based on their financial and transactional behavior using Machine Learning.

---

## 🎯 Objective

* Analyze customer data and behavior
* Group customers into meaningful clusters
* Identify high-value and low-value customers
* Build a Machine Learning model for segmentation
* Deploy an interactive web application for real-time predictions

---

## 🧠 Machine Learning Model

* **Model Used:** K-Means Clustering
* **Type:** Unsupervised Learning

### 📊 Output

* Cluster-based segmentation
* Customer grouped into different categories (e.g., High Spender, Low Spender)

---

## ⚙️ Technologies & Tools

* Python 🐍
* Pandas
* NumPy
* Scikit-learn
* Joblib
* Streamlit (for web app)

---

## 🧪 Techniques Used

* Data Cleaning & Preprocessing
* Handling Missing Values (Mean / Mode Imputation)
* Feature Selection
* Encoding (Categorical Data)
* Feature Scaling (StandardScaler)
* K-Means Clustering
* Elbow Method (Optimal K Selection)
* Model Deployment

---

## 📊 Features Used

* Customer Gender
* Customer Location
* Customer Account Balance
* Transaction Amount

---

## 🚀 Web Application

An interactive web application is built using Streamlit where users can:

* Input customer details
* Get instant segmentation result
* Identify customer category
